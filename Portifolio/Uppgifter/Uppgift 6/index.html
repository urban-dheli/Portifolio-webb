HTML

STRUKTUR

<DOCTYPE!>

Denna deklaration specifierar vilken version av HTML som används i dokumentet.

<html>

Rotelementet i ett HTML-dokumend som innehåller andra element.

<head>

En behållare för metadata om dokumentet, såsom titeln, länkar till stilmallar och metataggar.

<title>

Definierar titeln på HTML-dokumentet, som visas i webbläsarens titelrad eller flik.

<style>

Används för att bädda in CSS (Cascading Style Sheets) i ett HTML-dokument.

<link>

Specifierar externa resurser, som CSS-filer, att användas i dokumentet.

<body>

Huvudinnehållsområdet i ett HTML-dokument där text, bilder och annat innehåll visas.

Sematiska element

<main>

Representerar Huvudinnehållet i dokumentet, och det bör endast finnas ett <main>-element i en HTML-sida
